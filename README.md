# CVMacther

**CVMatcher** is a web-based AI-powered decision support system developed as part of the *Project-Based Learning (PBL)* course in the **Informatics Engineering Study Program**, **Department of Informatics Engineering**, **Politeknik Negeri Batam**.

This system is designed to assist Human Resource (HR) teams in screening and evaluating job applicants automatically using **Semantic Similarity Analysis** between resumes (CVs) and job descriptions.

By leveraging Natural Language Processing (NLP) and SBERT (Sentence-BERT), CVMatcher helps companies perform candidate selection more efficiently, objectively, and accurately.

# 📌 About The Project

Recruitment processes in many companies are still performed manually, where HR staff need to review CVs one by one. This process is time-consuming, inefficient, and prone to subjective bias.

In addition, traditional keyword-based filtering systems often fail to understand contextual meaning, causing potentially qualified candidates to be overlooked simply because of different wording in their resumes.

CVMatcher was developed to solve these challenges through AI-powered semantic similarity analysis.

---

# ❗ Problems Identified

- Manual CV screening requires significant time and effort
- Candidate evaluation can be subjective and inconsistent
- Traditional keyword filtering lacks semantic understanding
- HR teams struggle to process large numbers of applicants efficiently

---

# 💡 Solutions Offered

- Automated CV screening system
- Semantic similarity analysis using SBERT
- Objective candidate suitability scoring
- Candidate ranking system based on contextual relevance
- Faster and more data-driven recruitment decisions

---

# 🎯 Project Objectives

1. Develop a web-based platform for automated CV and job description analysis
2. Implement SBERT to measure semantic similarity between candidate profiles and job requirements
3. Generate objective ranking results to support HR decision-making processes
